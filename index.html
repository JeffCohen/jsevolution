JavaScript Code Evolution

This project uses a pretend stock exchange web service to demonstrate how to use JavaScript to build a serverless single-page application.

We start with an implementation in plain, classic JavaScript and slowly modernize the code in order to reveal the motivations behind each framework evolution.

(By "serverless" we just mean "someone else's server.")

FileNotes

stock_quote_jquery_1 Simplest possible jQuery
